Getting Started

Welcome to the MovieServer documentation page!

Installation

First, go ahead and visit the MovieServer Installer page and choose the right installer for your operating system.

The Windows installer is a .bat file, and the Mac and Linux installers are .sh files.

Windows

Download the MovieServer-Installer-Win.bat file and open it in File Explorer. If you see a Windows Defender warning (a dialog that appears before you open the file with a button named Run Anyway), click More Info..., then click Run Anyway. Then open ip.bat in the MovieServer folder. You will receive the same Windows Defender warning. Click More Info..., then Run Anyway.

Once the necessary tools are installed, and the MovieServer source code is installed, you should see a directory called MovieServer in your home directory.

Mac/Linux

Download the MovieServer-Installer-Mac.sh/MovieServer-Installer-Debian-Linux.sh file and open the Terminal app on your computer. Then type:

cd ~/Downloads/

Assuming installer .sh file is in Downloads folder.

Then execute the script.

# On Mac

./MovieServer-Installer-Mac.sh

# On Linux

./MovieServer-Installer-Debian-Linux.sh

Modification

Now that you've installed the dependencies and downloaded the source code, it is time to modify any IP addresses in files.

Windows

First, you need to find out your IP address on Windows. Open the Terminal app on your computer, then type:

ipconfig | findstr IPv4

You should see your IPv4 address. Copy-paste it and close Terminal.

Then open ip.bat in the MovieServer folder. If you see a Windows Defender warning (a dialog that appears before you open the ip.bat file with a button named Run Anyway), click More Info..., then click Run Anyway. You should now be prompted to enter an IP address.

Once you've entered your IP address, the program will look for all mentions of 192.168.254.137 in the MovieServer directory, and will replace them with the IP address you entered. Then, backup files with the .bak extension will be created in a folder called backups. This is done so that in the rare case of an error occurring and the modified files are damaged, you can refer to the backed-up versions and try to redo the process, or as a last resort, modify the program in any text editor or IDE.

Mac/Linux

First, you need to find out your IP address. Open the Terminal app on your computer, then type:

# On Mac

ipconfig getifaddr en0



You should see your IPv4 address. Copy-paste it and close Terminal.

# On Linux

hostname -I | awk '{ print $1 }'



You should see your IPv4 address. Copy-paste it and close Terminal.

Then execute ip.sh in Terminal.

# On Mac & Linux

~/MovieServer/ip.sh

You should be prompted for an IP address. Enter the IPv4 address you found earlier, and then you should be ready to go.

The program does the same thing that the Windows version does on Mac and Linux.

Starting the server

Now that you have finally modified all the IP addresses and installed all dependencies, we can now start the server. In the next chapter, we'll review how to add the movies to watch.

Windows

On Windows, type in:

cd %USERPROFILE%\MovieServer && node server.js

The servers should be available at your IP address with different ports printed once you start the servers.

Mac/Linux

On Mac/Linux, type in:

cd ~/MovieServer && node server.js

The servers should be available at your IP address with different ports printed once you start the servers.
